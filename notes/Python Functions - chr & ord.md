# Python Functions - chr & ord

[[Python Types]] [[Python Functions]]

* `ord(x)` 用来返回单个字符 %x% 的 ASCII 码
    * 字符串 --> 整数
* `chr(n)` 用来返回介于 0 到 255 之间的某 ASCII 码对应的字符
    * 整数 --> 字符串
    * 返回的字符串实为字符对应的 **2 位十六进制数**
        * 如 `chr(0)` 返回 `'\x00'`

/@ 大小写字母转换

```py
def swapcase(a):
    c = (ord(a)-ord('A'))*(ord(a)-ord('Z'))
    if c <= 0:
        b = chr(ord(a)-(ord('A')-ord('a')))
    else:
        b = chr(ord(a)+(ord('A')-ord('a')))
    return b
```
