# Python Functions - range

[[Python Functions]] [[Python Types]]

* 其既是一个函数也是一个序列数据类型 [[Python Types]], 返回一个 range 可迭代对象
    * 例如

        ```python
        >>> r = range(1,100)
        >>> r
        range(1,100)
        ```

## 参数

* `range([start], end, [step])`
    1. 起始值, 填整数. 默认为 0
    2. 终止值, 填整数
    3. 步长, 填**不为 0**整数. 默认为 1
* 返回的对象 **++包括起始值但不包括终止值++**
* 若 $(end-start)\cdot step \leqslant 0$ 则 `list(range(start, end, step)) = []`
