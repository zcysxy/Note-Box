# Python Functions - list

[[Python Functions]]

* 以作为参数的元组, range 对象, 字符串或其他类型的可迭代对象类型的数据新建一个[[Python Types - List|列表]]

/@

```python
>>> a = list((3,5,7,9,11)) # (3,5,7,9,11) 是一个元组
>>> a
[3, 5, 7, 9, 11]
>>> list(range(1,10,2)) # range(1,10,2) 是一个 range 对象
[1, 3, 5, 7, 9]
>>> list(‘hello world’)  # ‘hello world’ 是一个字符串
['h', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd']
>>> list()    # 创建空列表
```
